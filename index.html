Welcome to Micro Movinghead Project

The Micro Movinghead project is a project that combines Arduino coding, PCB, simple electronics and 3D printed parts to make a simple DMX controlled Movinghead which is as small as possible and therefor cheap.

Movie of the first version Movie of the strobe function of the first version

Design goals

This project has 2 main goals, first the final design should be a complete package and secondly the Movinghead should be as small a possible.

Spin-offs

This project also gives a framework on which different opensource lighting can be designed, for instance larger movingheads, scans, colorchangers.
